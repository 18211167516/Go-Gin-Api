1、项目说明

[!NOTE] 该项目为开源项目[Go-Gin-Api]项目文档

  • 用Gin快速搭建基础restful风格API，Gin文档是一个go语言编写的Web框架
  • 数据校验 go-playground github gin框架内置，已支持错误信息转中文
  • 数据库：采用MySql(5.6.44)版本，使用gorm 文档实现对数据库的基本操作。
  • API文档：使用Swaggergithub构建自动化文档。
  • 热重启HotStart github,我自己写的一个
  • 配置文件 使用viper github解析
  • 日志：使用zap github实现日志记录。
  • RBAC权限：使用casbin文档实现
  • 安全Cookie:使用encryptgithub实现加密解密
